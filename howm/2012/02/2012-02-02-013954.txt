= [memo][vim] vimチートシート
*モード*
i        カーソルの前で挿入モードへ
a        カーソルの後ろから挿入モードへ
A        カーソルの行末から挿入モードへ
o        カーソルの行の下に新しい行を追加し挿入モードへ
O        カーソルの行に新しい行を追加し挿入モードへ
R        置き換え(上書き)モードへ
r        一文字のみの置き換え(上書き)モードへ
s        カーソルのある１文字を変更
S        行全体を置き換え
cw       カーソル位置の単語を変更 (削除後挿入モード)
c^       カーソル位置から行頭までを変更 (削除後挿入モード)
c$       カーソル位置から行末までを変更 (削除後挿入モード)
ESC      編集モードへ
Ctrl+C   編集モードへ
Ctrl+[   編集モードへ

*移動*
h        ←
j        ↓
k        ↑
l        →
gj       表示行上カーソルを下に一字移動
gk       表示行上カーソルを上に一文字移動
Ctrl+f   １ページ進む
Ctrl+b   １ページ戻る
Ctrl+d   半ページ進む
Ctrl+u   半ページ戻る
$        行末に移動
^/0      行頭に移動
+        次行の先頭へ移動
-        前行の先頭へ移動
(        前の文章の先頭に移動
)        次の文章の先頭に移動
{        前の段落の先頭に移動
}        次の段落の先頭に移動
gg       ファイルの先頭へ移動
[n]G     n 行目に行く (省略でファイルの末尾へ移動)
H        画面最上に戻る
M        画面中央行へ移動
L        画面最下に進む
Ctrl+e   １行スクロールダウン
Ctrl+y   １行スクロールアップ
z<enter> カーソル位置が画面のトップｎなるようにスクロール
zz       カーソルが画面中になるようにスクロール
w        次の単語に移動
b        前の単語に移動
e        単語の最後に移動
f(文字)  カーソルがある行のその(文字)に移動
F(文字)  カーソルがある行のその(文字に)移動 (逆向き)
t(文字)  カーソルがある行のその(文字)の手前まで移動
T(文字)  カーソルがある行のその(文字)の手前まで移動 (逆向き)
%        対応する括弧に移動
Ctrl+g   ファイルの中野現在の場所を知る

*編集*
J        行の連結
D        カーソル位置から行末まで削除
yy / Y   カーソルのある行をコピー
dd       カーソルのある行を切り取り
x        カーソルの下にある文字を削除
X        カーソルの左の文字を１文字削除
p        カーソルの後ろに貼り付け
P        カーソルの前に貼り付け
.        直前の変更を繰り返す
u        アンドゥ
Ctrl+r   リドゥ

*ファイル*
:q       終了(:q!で保存せずに強制終了)
:w file  file という名前で保存 (省略の場合は上書き保存)
ZZ       保存して閉じる
:#,# w   #行目から#行目までを保存
:set fileencoding=utf-8    文字コードの指定
:cd path カレントディレクトリを path に設定
:pwd     カレントディレクトリを表示
:help index    vim コマンドの全目次を見る
gf       カーソル位置にある文字列をファイル名とみなして開く

*検索と置換*
/word    word を検索 (n で次の候補、N で前の候補) (/ を検索するときは \/)
?word    word を逆方向に検索
/word/+n 検索もジュイ列の n 行後へ移動
*        カーソル位置にある単語を検索
#        カーソル位置にある単語を検索 (逆方向)
:%s/[old]/[new]/g   単語の置換 ([old] を [new] へ置換)
                     (gc にすると置換えの度に確認を求める)
:set ignorecase    大文字小文字を無視
:set noignorecase  大文字小文字を区別する
:set hlsearch    ヒット結果のハイライト表示

*領域選択*
v        領域選択スタート
Ctrl+v   矩形選択スタート
Shift+v  行選択スタート
gv       最後の選択範囲を選択肢なおす
y        コピー
d        切り取り
=        選択領域をオートインデント
(矩形選択した状態で)I#<Esc>        選択範囲の行頭に#が追加される
(矩形選択した状態で)$A.cpp<Esc>    選択範囲の行末に.cppが追加される

*タグ移動*
:tag function    関数 function へ移動
:tags    タグリストを表示
:tag     タグリストの先頭へ移動
Ctrl+]   カーソル業の関数定義位置へ移動
Ctrl+T   直前のタグへ戻る

*ウィンドウ*
:split (:sp)   画面を上下に分割する
:vsplit(:vs)   画面を左右に分割する
:close   ウィンドウを閉じる
:new filename    新規ウィンドウ作成 (垂直方向)
:vnew filename   新規ウィンドウ作成 (水平方向)
:e filename      今いるウィンドウにファイルを開く
:q       ウィンドウの削除
:hide    ウィンドウを隠す (バッファには残る)
Ctrl+w + ウィンドウを拡大
Ctrl+w - ウィンドウを縮小
Ctrl+w _ 現在のウィンドウを最大化
Ctrl+w > ウィンドウの幅を１行増やす
Ctrl+w < ウィンドウの幅を１行減らす
Ctrl+w = ウィンドウのサイズを同じにする
Ctrl+w w 別のウィンドウへ移動
Ctrl+w k 上のウィンドウへ移動
Ctrl+w j 下のウィンドウへ移動
Ctrl+w h 左のウィンドウへ移動
Ctrl+w l 右のウィンドウへ移動
Ctrl+w r 左右/上下のウィンドウを入れ替え
Ctrl+w c ウィンドウを閉じる
Ctrl+w o 他のウィンドウを閉じる

*buffer (バッファ管理)*
:ls      バッファ一覧を表示
:b buffnum    今いるウィンドウに特定のバッファを呼び出す
:bd      現在のバッファを閉じる
:bn      次のバッファに移動
:bp      前のバッファに移動

*tab (タブ操作)*
:tabnew      新しいタブページを開く
:tabn / gt   次のタブページに移動
:tabp / gT   前のタブページに移動
:tabc        現在編集しているタブページを閉じる
:tabfir      先頭のタブページに移動
:tabl        最後のタブページに移動
:tabd        カレントウィンドウのすべてのタブページでコマンド実行
:tabd %s/hoge/hoge1/g   カレントウィンドウのすべてのタブページでhogeをhoge2に置換する
*register (レジスタ管理)*
"ay      選択範囲をレジスタ a に保存
"ayy     今いる行をレジスタ a に保存
"ap      レジスタ a の内容をカーソル位置にペースト
"*p      クリップボードの内容をカーソル位置にペースト*
:reg     レジスタに格納されている情報を一覧表示

*mark (マーク)*
ma       現在のカーソル位置をマーク名 a に保存
'a       マーク名 a の位置に移動
:marks   マークの一覧を表示する

*folding (折りたたみ)*
zf       選択領域を折りたたむ
スペース 折りたたみを展開する
zo       折りたたみを展開する
zd       折りたたみを消す
za       折りたたみを開閉する
zi       折りたたみを無効にしたり有効にしたりする
zj       次の折りたたみに移動
zk       前の折りたたみに移動
mk (:mkview)    折りたたみの保存 
:loadview        折りたたみの読み込み

*word completion (単語補完)*
Ctrl+p   単語補完 (前方検索)
Ctrl+n   単語補完 (後方検索)

*recording (操作記憶)*
qa       操作の記録を開始し、レジスタ a に保存する
qA       レジスタ a に操作の記録を追加する (大文字の場合は追記)
q        操作の記録を終了する
@a       レジスタ a に保存された操作を再生する
@@       最後に実行したレジスタを再生する
5@a      レジスタ a に保存された操作を5回再生する

*外部コマンドとの連携*
:lcommand    外部コマンド実行 (領域指定すると出力も取込)
:rlcommand   外部コマンド実行 (カーソル位置へ出力挿入)

*diff操作*
:vertical diffsplit tmp1 tmp2    tmp1とtmp2のdiffを表示
:diffoff  diff表示を解除
:diffupdate	diff    表示を更新する
do        カーソル位置の差分を他方からマージする
dp        カーソル位置の差分を他方にマージする
]c        次の差分へ移動
[c        前の差分へ移動

*そのた*
di"       「""」の中身を削除する
da"       「""」ごと削除する
di(       「()」の中身を削除する
da(       「()」ごと削除する

*コマンドラインモード*
Ctrl-R "  レジスタの値（dd/yyで削除/コピーした値)を貼り付け
Ctrl-R +  クリップボードの値を貼り付け
