= [memo][vim] opsplorer.vim : ツリー型のファイルエクスプローラー

操作方法の一覧です。

                            カーソルの移動に関わる操作                            
 +-------------------------------------------------------------------------------+
 |     キー      |     カーソルの位置      |                効果                 |
 |---------------+-------------------------+-------------------------------------|
 |Enterキーか    |ディレクトリ名の前の「+  |指定のディレクトリを開く。           |
 |Spaceキー      |」の文字の上             |                                     |
 |---------------+-------------------------+-------------------------------------|
 |Enterキーか    |ディレクトリ名の前の「-  |指定のディレクトを閉じる。           |
 |Spaceキー      |」の文字の上             |                                     |
 |---------------+-------------------------+-------------------------------------|
 |Enterキーか    |                         |ファイルエクスプローラーのベースディ |
 |Spaceキー      |ディレクトリ名の上       |レクトリを指定のディレクトに変更する |
 |               |                         |。                                   |
 |---------------+-------------------------+-------------------------------------|
 |Enterキーか    |ファイルエクスプローラー |親ディレクトリに移動する。           |
 |Spaceキー      |のベースディレクトリの上 |                                     |
 |---------------+-------------------------+-------------------------------------|
 |Shiftキーを押  |---                      |1つ次のディレクトリにカーソルを移動  |
 |しながら上キー |                         |する。                               |
 |---------------+-------------------------+-------------------------------------|
 |Shiftキーを押  |---                      |1つ前のディレクトリにカーソルを移動  |
 |しながら下キー |                         |する。                               |
 |---------------+-------------------------+-------------------------------------|
 |qキー          |---                      |ファイルエクスプローラーを閉じます。 |
 +-------------------------------------------------------------------------------+
                        ファイルを開く時の挙動に関わる操作                        
 +-------------------------------------------------------------------------------+
 |  キー   |カーソルの |                          効果                           |
 |         |   位置    |                                                         |
 |---------+-----------+---------------------------------------------------------|
 |Enterキ  |ファイルの |指定のファイルを開きます。ファイルエクスプローラーは閉じ |
 |ー       |上         |ます。                                                   |
 |---------+-----------+---------------------------------------------------------|
 |Spaceキ  |ファイルの |指定のファイルを開きます。ファイルエクスプローラーは開か |
 |ー       |上         |れたままになります。                                     |
 +-------------------------------------------------------------------------------+
                            ファイルの表示に関わる操作                            
 +-------------------------------------------------------------------------------+
 |   キー    |                               効果                                |
 |-----------+-------------------------------------------------------------------|
 |H          |隠しファイルの表示非表示を切り替えます。デフォルトは非表示です。   |
 |(Shift-h)  |                                                                   |
 |-----------+-------------------------------------------------------------------|
 |M          |ファイルの絞り込み検索を行います。ワイルドカードが使用可能。デフォ |
 |(Shift-m)  |ルトは「*」です。                                                  |
 +-------------------------------------------------------------------------------+
                      ファイルの操作                      
 +-------------------------------------------------------+
 |    キー     |カーソルの位置 |          効果           |
 |-------------+---------------+-------------------------|
 |N (Shift-n)  |ファイル名の上 |ファイル名を変更します。 |
 |-------------+---------------+-------------------------|
 |C (Shift-c)  |ファイル名の上 |ファイルをコピーします。 |
 |-------------+---------------+-------------------------|
 |D (Shift-d)  |ファイル名の上 |ファイルを削除します。   |
 |-------------+---------------+-------------------------|
 |O (Shift-o)  |ファイル名の上 |ファイルを移動します。   |
 +-------------------------------------------------------+

設定

初期設定を変更したい場合は、下の設定をvimの設定ファイルに書き込んで下さい。

 +-------------------------------------------------------------------------------+
 |                             |               |デ |                             |
 |                             |               |フ |                             |
 |         オプション          |     説明      |ォ |           設定例            |
 |                             |               |ル |                             |
 |                             |               |ト |                             |
 |                             |               |値 |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |1に設定すると  |   |                             |
 |                             |マウスでファイ |   |                             |
 |                             |ルを1回クリッ  |   |                             |
 |                             |クした時にファ |   |:let s:single_click_to_edit= |
 |s:single_click_to_edit       |イルを開くよう |0  |1                            |
 |                             |になる。デフォ |   |                             |
 |                             |ルトではダブル |   |                             |
 |                             |クリック時に開 |   |                             |
 |                             |く。           |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |ファイルの絞り |   |                             |
 |                             |込みの検索条件 |   |:let s:file_match_pattern=   |
 |s:file_match_pattern         |。デフォルトは |*  |"*.c"                        |
 |                             |「*」で全ファ  |   |                             |
 |                             |イルの表示。   |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |1に設定すると  |   |                             |
 |                             |隠しファイルを |   |                             |
 |s:show_hidden_files          |表示する。デフ |0  |:let s:show_hidden_files=1   |
 |                             |ォルトでは隠し |   |                             |
 |                             |ファイルを表示 |   |                             |
 |                             |しない。       |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |ファイルエクス |   |                             |
 |                             |プローラーを表 |   |                             |
 |s:split_vertical             |示する時にウィ |1  |:let s:split_vertical=0      |
 |                             |ンドウを縦に分 |   |                             |
 |                             |割するか、横に |   |                             |
 |                             |分割するか。   |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |ファイルエクス |   |                             |
 |s:split_width                |プローラを表示 |16 |:let s:split_width=32        |
 |                             |する横幅       |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |ファイルエクス |   |                             |
 |s:split_minwidth             |プローラーの横 |1  |:let s:split_minwidth=16     |
 |                             |幅の最小値     |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |ファイルエクス |   |                             |
 |                             |プローラーで色 |   |                             |
 |                             |を使うか使わな |   |                             |
 |s:use_colors                 |いか。デフォル |1  |:let s:use_colors=0          |
 |                             |トは1で使う設  |   |                             |
 |                             |定になっている |   |                             |
 |                             |。             |   |                             |
 |-----------------------------+---------------+---+-----------------------------|
 |                             |マウスでファイ |   |                             |
 |                             |ル名を選択した |   |:let                         |
 |s:close_explorer_after_open  |時にファイルエ |0  |s:close_explorer_after_open= |
 |                             |クスプローラー |   |1                            |
 |                             |を閉じるか、閉 |   |                             |
 |                             |じないか。     |   |                             |
